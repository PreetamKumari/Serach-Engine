If you have not installed XAMPP,

1. Install xampp on your PC
2. Go to the drive you have installed xampp on -> xampp -> htdocs
3. unzip the folder in htdocs

Assuming that you have already set up XAMPP, open Xampp

Start the database server ("MySQL Database")
Start the PHP environment ("Apache Web Server")

4. In your browser open "http://localhost/phpmyadmin/"
5. On the left side goto new button and click -> this will create a new database
6. Name the database "google_new"
7. Goto import -> choose file -> import the sql file in the zip folder -> go
8. In your browser open "http://localhost/Task3_SearchEngine/index.php"

About the project,
-The page allows you to search from the crawled websites and saves them in the database.
Example search keywords - Ronaldo, messi, neymar, instagram, etc.
-A configuration button on the top right corner allows you to add websites to the databse
